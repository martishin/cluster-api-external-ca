# External CA PoC Assets

The primary testing and build guide is at repository root:

- `README.md`

This folder contains:

- `env/`: kind management cluster config
- `manifests/`: self-signed and external-ca cluster manifests
- `scripts/`: end-to-end PoC automation
- `capi-patches/`: patch bundle applied to upstream CAPI for local builds
- `../out/`: runtime artifacts generated by scripts
